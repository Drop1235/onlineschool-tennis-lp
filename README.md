# growwith-tennis-lp

Growith Online School（Tennis）のLP（静的HTML）です。

## 開き方（ローカル）

- `index.html` をブラウザで直接開く
- もしくは簡易サーバで起動（例）
  - `python -m http.server 5173`
  - ブラウザで `http://127.0.0.1:5173/`

## 編集するファイル

- `index.html`
